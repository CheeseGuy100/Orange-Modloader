🍃 Features

🌐

Modrinth Integration

Browse thousands of mods directly from Modrinth. Search, filter by Java or Bedrock, and install with one tap.

☕

Java & Bedrock Support

Works with both Minecraft Java Edition mods and Bedrock addons. Filter by your version and install instantly.

📱

Android & Windows

Available on Android APK and Windows MSI. Manage your mods anywhere, anytime.

🔓

Free & Open Source

MangoLoader is MIT licensed and completely free. No ads, no paywalls, no tracking. Ever.

🔍

Smart Search

Find mods instantly with the built in search bar. Filter by name, author or category.

🧀

Easter Eggs

Tap the version number in settings 5 times... you'll see 😭
